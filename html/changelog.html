12 July 2020

carlopault updated:

  • Farts now spawn miasma. Superfarts spawn ~4 times the miasma than normal farts. (Mostly coded by Yoyobatty)
  • The Spacestar Ordering Book Catalog no longer tells you it is anchored with bolts when you examine it, as that is pointless.
  • I fixed a bug where you were not given the crafting recipe to make a tier 3 obelisk when constructing the tier 2 obelisk.

26 June 2020

carlopault updated:

  • Magic soap is now an actual item
  • Slightly tweaked how the magical inscripture worked.
  • The monster cube crafting recipe has been changed to something that I think should show up properly in the crafting menu, unlike the current recipe.
  • swapped the ink prices of The Mark of Cain and Grabar: A Slow Mindkill in the Spacestar
  • Removed shoes from the magical soap recipe.
  • Icon for magic soap

06 June 2020

carlopault updated:

  • The library now has a computer that can be used to print crafting books: Spacestar Ordering Book Catalog. The computer operates on special ink cartridges that the curator spawns with (1 cartridge spawns on the computer).
  • Added many new crafting books (books that give you crafting recipes)
  • Added new items to the game that can be crafted via the crafting books obtainable through the Spacestar Ordering Book Catalog.
  • Removed the crafting book beacon from the library backroom as it no longer has a purpose.
  • Cargo can now order Spacestar-brand ink cartridges.
  • The curator has been given more things to do now (duh).
  • New sounds for one of the new items.
  • New sprites for the new stuff (Sprites by Mr. Epic Sax and Punkinguy).
  • Changed the obj pathing of crafting granter books. All of this was coded with the help of Neo.

19 May 2020

CrashMSO updated:

  • Readded headslugs to the gold slime pool (Xenolings)
  • Readded statues to the gold slime pool

yoyobatty updated:

  • Added the shocker and throwing turret for adminbus fun!
  • Acid processing no longer adds 100's of overlays by accident
  • Added a random reagent beaker, self explanitory, for badmins only
  • Huge beakers now spawn in chemical closet lockers
  • Huge beakers (500u) are now in the game as a minor chemistry buff
  • Vapor now looks MORE like vapor
  • Cleaned up some old code related to how chemical reagent dispensers worked
  • Fixed sec wardrobe going invisible when powered off

15 May 2020

yoyobatty updated:

  • Math defines have been replaced with their built in procs
